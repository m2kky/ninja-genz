## Overview

**Key Features:**

- Review deliverables
- Approve or request revisions
- Add comments
- Download files

## Security (Client Access)

**RLS Policy:**
```sql
CREATE POLICY "Clients view own projects only"
ON tasks FOR SELECT
USING (
  EXISTS (
    SELECT 1
    FROM projects p
    JOIN clients c ON c.id = p.client_id
    WHERE p.id = tasks.project_id
      AND c.id IN (
        SELECT client_id FROM client_users WHERE user_id = auth.uid()
      )
  )
);
```

**Read-Only Enforced:**
- Clients CANNOT create/delete tasks
- Clients CANNOT assign team members
- Clients CAN ONLY approve or request revisions
