To achieve the "Ninja Gen Z" aesthetic (Cyberpunk, Neon, High-Performance), we will leverage these specialized libraries.

## 1. Magic UI ðŸª„
**The "Playful" Vibe** - Perfect for micro-interactions and "wow" moments.
- **Website:** [magicui.design](https://magicui.design)
- **Key Components:**
  - `AnimatedSubscribeButton`
  - `BentoGrid` (Essential for dashboards)
  - `Marquee` (For client logos)
  - `Globe` (For landing page hero)
- **Install:** `npx shadcn-ui@latest add https://magicui.design/r/[component]`

## 2. Aceternity UI ðŸŒŒ
**The "Premium" Vibe** - High-end, cinematic animations powered by Framer Motion.
- **Website:** [ui.aceternity.com](https://ui.aceternity.com)
- **Key Components:**
  - `TracingBeam` (For timeline tracking)
  - `BackgroundBeams` (For auth pages)
  - `Sparkles` (For success states)
  - `TypewriterEffect` (For Sanad AI typing)
- **Install:** Copy-paste components (Cn command available for some).

## 3. Specialized Shadcn Extensions
- **Solless UI:** Beautifully animated extensions for Shadcn.
- **Plate:** Notion-style rich text editor built on Shadcn (Upgrade from Tiptap if needed).
- **Vaul:** Drawer component for mobile-native feel.

## 4. Icons & Assets
- **Lucide React:** Standard icons.
- **Heroicons:** Alternative outlined icons.
- **Simple Icons:** For brand logos (Facebook, Instagram, etc.).
